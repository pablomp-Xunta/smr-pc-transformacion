# Ideas de Diseño — Tu PC, Mil Posibilidades

## Enfoque elegido: "Terminal Educativo"

**Movimiento de Diseño:** Estética de laboratorio informático / terminal oscuro con toques de interfaz de administración de sistemas.

**Principios Fundamentales:**
1. Fondo oscuro tipo sala de servidores — transmite profesionalidad técnica
2. Tipografía mixta: `JetBrains Mono` para títulos y código, `Source Sans 3` para cuerpo
3. Tarjetas de sistema que simulan paneles de administración reales
4. Jerarquía visual clara mediante color y espaciado generoso

**Filosofía de Color:**
- Fondo principal: `#0d1117` (negro GitHub / sala oscura de servidores)
- Fondo secundario: `#161b22` (paneles elevados)
- Acento primario: `#00d084` (verde terminal — éxito, actividad)
- Acento secundario: `#58a6ff` (azul eléctrico — información, enlaces)
- Texto principal: `#e6edf3` (blanco suave)
- Texto secundario: `#8b949e` (gris muted)

**Paradigma de Layout:**
- Navegación lateral fija con iconos de cada sistema
- Contenido principal en scroll vertical con secciones diferenciadas
- Hero asimétrico con texto a la izquierda y visualización técnica a la derecha

**Elementos Distintivos:**
- Badges de "dificultad" con colores semáforo
- Bloques de código con syntax highlighting simulado
- Indicadores de requisitos de hardware en formato tabla técnica

**Filosofía de Interacción:**
- Hover effects en tarjetas con borde iluminado
- Transiciones suaves al navegar entre secciones
- Animaciones de entrada escalonadas (stagger 50ms)

**Tipografía:**
- Títulos: JetBrains Mono Bold — evoca terminal, código, precisión técnica
- Cuerpo: Source Sans 3 Regular — legibilidad óptima para texto educativo largo
- Código/etiquetas: JetBrains Mono — consistencia técnica
